01. Styling Links <a>

Links (<a>) are the backbone of navigation. Style them with padding, borders, and hover effects to create cyberpunk buttons that glow and respond to mouse movement. 
<a href="#" class="cyber-link">Neon Link</a>
<a href="#" class="cyber-link">Hover Me</a>

/* CSS */
.cyber-link {
    color: #00ffff;
    padding: 8px 16px;
    border: 1px solid rgba(0, 255, 255, 0.5);
    border-radius: 20px;
    transition: all 0.3s ease;
}

.cyber-link:hover {
    background: rgba(0, 255, 255, 0.2);
    box-shadow: 0 0 25px rgba(0, 255, 255, 0.8);
}

02. Styling Lists <ul> <ol>

Lists are perfect for menus, features, and navigation. Remove default bullets with list-style: none, then add custom neon bullets or numbers with pseudo-elements. 
<ul class="cyber-list">
    <li><a href="#">Student System</a></li>
    <li><a href="#">Gamified App</a></li>
    <li><a href="#">Neon Portfolio</a></li>
</ul>

/* CSS */
.cyber-list { list-style: none; }
.cyber-list li {
    background: rgba(10, 15, 50, 0.8);
    padding: 20px;
    border-left: 4px solid #ff00ff;
    position: relative;
}
.cyber-list li::before {
    content: '▸';
    color: #00ffff;
    position: absolute;
    left: 15px;
}

04. Mobile Responsive Navbar

On mobile, the navbar collapses into a hamburger menu. Use media queries to stack the menu vertically and show a toggle button. The menu slides down with smooth animations. 
@media (max-width: 768px) {
    .nav-menu {
        position: absolute;
        top: 100%;
        left: 0;
        right: 0;
        flex-direction: column;
        background: rgba(0, 0, 20, 0.98);
        transform: translateY(-100%);
        opacity: 0;
    }
    
    .nav-menu.active {
        transform: translateY(0);
        opacity: 1;
    }
}
Add a hamburger button with JavaScript or CSS-only solutions. This creates professional mobile navigation that works perfectly for your student portfolio sites.

05. Pro Tips & Best Practices

  • Always remove default list styles:
    ul { list-style: none; padding: 0; margin: 0; }
    Clean starting point for custom navbars and menus.
  • Use flexbox for horizontal nav:
    .nav-menu { display: flex; gap: 30px; }
    Perfect spacing between menu items.
  • Mobile-first responsive:
    Always plan for phones first. Desktop navbars break on mobile without media queries.
  • Hover effects = engagement:
    Links that glow, lift, and change color feel alive and cyberpunk.
