Understanding Dimensions

Lesson 7 ‚Ä¢ Topic 1
CSS elements have width and height that control their size. Use fixed sizes (pixels) or fluid sizes (% or viewport units) for responsive design!

Resize browser to see differences:

FIXED 250px
FLUID 70%

Width & Height Properties

Use width and height to set exact sizes. Pixels = fixed, % = relative to parent, vh/vw = viewport size.
div.fixed {
    width: 300px;
    height: 150px;
}
div.fluid {
    width: 60%;
    height: 20vh;
}

Max-Width Magic

max-width stops elements from getting too wide! Perfect for responsive containers that scale but stay readable.
.container {
    max-width: 800px;
    width: 90%;
    margin: auto;
}
Resize me! Max 500px wide

Fixed vs Fluid Layouts

FIXED: pixels (consistent but not responsive)
FLUID: % or vw/vh (adapts to screen)
RESPONSIVE: max-width + % (best of both!)
.fixed { width: 400px; }
.fluid { 
    width: 70%; 
    max-width: 1000px; 
}

Viewport Units (vh/vw)

vh = 1% of viewport height, vw = 1% of viewport width. Perfect for full-screen elements!
header {
    height: 100vh;  /* Full screen */
    width: 100vw;   /* Full width */
}
.box {
    width: 50vw;    /* 50% screen width */
    height: 25vh;   /* 25% screen height */
}
