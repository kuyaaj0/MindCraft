CSS Width, Height & Max-Width

Lesson 7 - Fixed vs. Fluid Layout Basics

01. Understanding Dimensions

In CSS, elements have dimensions ‚Äî width and height ‚Äî that determine their size. These can be fixed (exact values like pixels) or fluid (responsive using percentages or viewport units).

02. The width & height Properties

These properties define how wide and tall an element should be. They can use:
  • px ‚Üí fixed size, not responsive.
  • % ‚Üí relative to parent container (fluid).
  • vh/vw ‚Üí relative to viewport height and width.
div.fixed-box {
    width: 300px;
    height: 150px;
}

div.fluid-box {
    width: 60%;
    height: 20vh;
}
300px Fixed Box
60% Fluid Box

03. The max-width Property

max-width sets the maximum width an element can stretch to. It‚Äôs a powerful tool for creating responsive designs where content scales but doesn‚Äôt get too wide. 
div.container {
    max-width: 800px;
    width: 90%;
    margin: auto;
}
Try resizing your browser ‚Äî this container stops growing at 800px wide!

04. Fixed vs. Fluid Layouts

- Fixed layouts use exact pixel sizes ‚Äî consistent but not responsive.
- Fluid layouts use relative units (%, vw, vh) and adapt to screens.
- Responsive layouts combine both, setting max-width for control. 
.fixed {
    width: 400px;
}

.fluid {
    width: 70%;
    max-width: 1000px;
}
Fixed width (400px)
Fluid (70%) with max-width = 1000px
