CSS Selectors

Lesson 3 - Targeting Elements Efficiently

01. What Are CSS Selectors?

A CSS selector tells the browser which HTML elements to style. It’s how we target parts of a web page — like paragraphs, buttons, or titles — and apply specific designs to them.

02. Element Selector

Targets elements directly by name — for example p, h1, or div. All matching elements will share the same style. 
/* Applies style to ALL paragraphs */
p {
    color: cyan;
    font-size: 18px;
}

Page Title

This paragraph styles every <p> element the same way.

All paragraphs share this cyan color and font size.

03. Class Selector (.)

Classes are used when you want to style multiple elements in the same way. They start with a period (.). Apply them in HTML using class="name". 
/* CSS */
.special {
    color: hotpink;
    text-shadow: 0 0 10px #ff00ff;
}

/* HTML */
<p class="special">This text uses the "special" class.</p>

This text uses the “special” class!

04. ID Selector (#)

IDs are unique identifiers — used for one element only. They start with a hash symbol (#) and are applied in HTML using id="name". 
/* CSS */
#unique {
    background-color: rgba(0,255,255,0.15);
    border: 2px solid cyan;
    padding: 15px;
}

/* HTML */
<div id="unique">I have a unique ID!</div>
I have a unique ID!

05. Element vs Class vs ID

- Element selector: Styles all tags of that type.
- Class selector: Styles a group of elements sharing the same class.
- ID selector: Styles one specific element only. 
/* Example CSS */
h2 { color: cyan; }
.special { color: hotpink; }
#unique { color: yellow; }

This is an Element Selector (h2)

This is a Class Selector (.special)

This is an ID Selector (#unique)
