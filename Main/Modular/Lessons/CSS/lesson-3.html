CSS Selectors

Lesson 3 â€¢ Targeting Elements
CSS selectors tell browser WHICH HTML to style! Target paragraphs, buttons, titles with specific designs.

3 Selector Types:

Element Selector

Lesson 3 â€¢ Topic 2
Targets ALL elements by tag name. p = all paragraphs, h1 = all H1s.
p {
  color: cyan;
  font-size: 18px;
}

ALL <p> tags get cyan!

Every paragraph matches!

Class Selector (.dot)

Lesson 3 â€¢ Topic 3
Use . + class name. Multiple elements can share same class! Add class="special" in HTML.
.special {
  color: hotpink;
  text-shadow: 0 0 10px #ff00ff;
}

/* HTML */
<p class="special">Hotpink text!</p>

.special class demo!

ID Selector (#hash)

Lesson 3 â€¢ Topic 4
# + ID name. UNIQUE - only ONE element per page! Use id="unique" in HTML.
#unique {
  background: rgba(0,255,255,0.15);
  border: 2px solid cyan;
  padding: 15px;
}

/* HTML */
<div id="unique">Unique!</div>
#unique ID demo!

Element vs Class vs ID

Lesson 3 â€¢ Summary
Element: ALL matching tags (p, h1, div)
Class: GROUP of elements (.special)
ID: ONE unique element (#unique) 
h2 { color: cyan; }
.special { color: hotpink; }
#unique { color: yellow; }

h2 = Element

.special = Class

#unique = ID
