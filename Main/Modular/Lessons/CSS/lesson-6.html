01. What is the Box Model?

Every HTML element is a box. The CSS Box Model describes how much space an element takes up and how its different layers ‚Äî content, padding, border, and margin ‚Äî affect layout and spacing.

03. Padding

Padding is the inner space between the content and the border. It adds breathing room inside the box. 
div {
    padding: 20px;
    background-color: #111144;
}
üåê The padding pushes the content inward.

04. Border

The border wraps around the padding and content. It creates outlines or frames around the box.

    border: 4px solid #00ffff;
    padding: 20px;
}
This box has a glowing cyan border.

05. Margin

Margin is the outer space outside the border. It separates one element from another ‚Äî like invisible walls between boxes. 
div {
    margin: 30px;
    border: 2px solid #ff00ff;
}
üëæ Margins create space outside this box.

06. box-sizing

By default, width & height only include the content. The box-sizing property changes that.
  • content-box: (default) width = content only.
  • border-box: width includes padding & border ‚Äî easier for responsive design.
.default-box {
    box-sizing: content-box;
}

.border-box {
    box-sizing: border-box;
}
