01. Margin vs Padding

Both margin and padding control spacing ‚Äî but in different ways:
  • Margin = space outside the element (creates gaps between elements)
  • Padding = space inside the element (pushes content away from borders)
div {
  margin: 30px;
  padding: 20px;
}
Margin adds space outside the box. Padding adds space inside.

02. Centering Blocks

To center a block element horizontally, set its left and right margins to auto. You‚Äôll also need to define a width so the element knows how wide it should be. 
.center {
  width: 300px;
  margin: auto;
}
Centered Block (using margin: auto)

03. Stacking Sections

Use consistent top and bottom margins or padding between sections to create a pleasant visual flow. This makes the layout breathable and easier to read. 
section {
  margin-bottom: 40px;
  padding: 30px 20px;
}

04. Common Layout Patterns

Margin and padding are essential for these layout techniques:
  • Centered Card: margin: auto + width
  • Stacked Sections: margin-bottom for spacing
  • Even Padding: padding adds comfort inside elements
