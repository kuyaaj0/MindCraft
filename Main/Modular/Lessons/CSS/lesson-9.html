Margin vs Padding

Lesson 9 ‚Ä¢ Topic 1
Margin = space OUTSIDE elements (between boxes). Padding = space INSIDE elements (content to border).
MARGIN (outside)
PADDING (inside)

Centering Blocks

Use margin: auto + fixed width to center blocks horizontally!
.center {
  width: 300px;
  margin: auto;
}
Centered with margin: auto

Stacking Sections

Use margin-bottom or margin-top to space stacked sections!
section {
  margin-bottom: 40px;
  padding: 30px 20px;
}

Shorthand Values

Use shorthand: margin: 20px (all sides), margin: 10px 20px (top/bottom left/right).
margin: 20px;          /* All sides */
margin: 10px 20px;     /* Top/bottom, left/right */
margin: 15px 30px 10px; /* Top, left/right, bottom */
padding: 20px 15px;    /* All around */
