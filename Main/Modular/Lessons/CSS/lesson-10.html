What Is Display?

Lesson 10 ‚Ä¢ Topic 1
display controls how elements flow: block = new line, inline = same line, inline-block = both + sizing!

Watch these change with display types:

Display Test Box

Block Elements

display:block = full width, new line, accepts width/height/margin.
div {
  display: block;
  width: 300px;
  background: #111144;
}
BLOCK 1 (full width)
BLOCK 2 (new line)

Inline Elements

display:inline = no line breaks, only content width, no height/width control.
span {
  display: inline;
}

Inline spans: INLINE A INLINE B INLINE C all same line!

Inline-Block Magic

display:inline-block = inline flow + block sizing (width/height work)!
div {
  display: inline-block;
  width: 120px;
}
Box A
Box B
Box C

Display: None

display:none = completely removes element + space!
#secret {
  display: none;
}
