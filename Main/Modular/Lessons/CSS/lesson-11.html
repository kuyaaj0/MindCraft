    top: 0;

06. ü•á z-index (Layer Stacking)

z-index = LAYER ORDER. Higher numbers appear on top. Only works on positioned elements (relative, absolute, fixed, sticky). 
.layer1 { z-index: 1; }
.layer2 { z-index: 2; }
.layer3 { z-index: 999; }
