02. font-size

font-size controls how large or small your text appears. You can use px (pixels), em (relative size), or rem for responsive scaling. 
h1 { font-size: 48px; }
p { font-size: 1.2em; }

03. line-height

line-height controls the space between lines of text. A higher value makes text easier to read, especially for long paragraphs. 
p {
    line-height: 1.8;
}

This paragraph has extra spacing between lines, making it more comfortable to read under neon light!

04. text-align

05. text-decoration

text-decoration adds visual styles like underlines or strikethroughs. Use it for links or emphasis. 
a {
    text-decoration: underline;
}
del {
    text-decoration: line-through;
}

06. font-weight

The font-weight property makes text bolder or lighter. Numeric values range from 100 (thin) to 900 (extra bold). 
h1 { font-weight: 900; }
p { font-weight: 300; }

07. Google Fonts Integration

You can import Google Fonts to make your designs stand out. Simply visit fonts.google.com, choose a font, and copy its @import or <link>. 
@import url('https://fonts.googleapis.com/css2?family=Poppins:wght@400;700&display=swap');

body {
    font-family: 'Poppins', sans-serif;
}

This sentence uses a Google Font for a clean and modern neon interface look.
