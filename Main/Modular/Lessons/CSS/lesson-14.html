Inline vs External

CSS + HTML â€¢ Topic 2
Inline = quick but messy. External = organized for big projects.
  • INLINE: <p style="color:pink"> (1 element only)
  • INTERNAL: <style>p { color: cyan; }</style> (1 page)
  • BEST: <link href="style.css"> (many pages)
