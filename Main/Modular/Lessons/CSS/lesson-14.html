01. Three Ways to Use CSS

CSS controls the design of the page: colors, fonts, spacing, layout. There are three main ways to connect CSS to HTML: inline, internal, and external (separate .css file). [web:10][web:4] 
<!-- INLINE CSS: style attribute on the tag -->
<h1 style="color:#ff00ff; text-align:center;">
    Neon Title
</h1>

<!-- INTERNAL CSS: inside <style> in the <head> -->
<head>
    <style>
        body {
            background-color: #000;
            color: #e0e0e0;
        }
    </style>
</head>

<!-- EXTERNAL CSS: separate file style.css -->
<head>
    <link rel="stylesheet" href="style.css">
</head>

INLINE STYLED TITLE

This line is styled directly on the element using the style attribute.

02. Inline vs Internal vs External

Inline CSS is written inside the HTML tag and affects only that element. Internal CSS is in a <style> tag in the head and styles the whole page. [web:4][web:10]
  • INLINE CSS (quick, but messy):
    <p style="color:#ff1493; font-size:18px;">Pink text.</p>
    Good for testing but hard to maintain when the project grows.
  • INTERNAL CSS (one page only):
    <style> p { color:#00ffff; } </style>
    Nice for small single-page demos or lessons.
  • BEST PRACTICE = EXTERNAL CSS:
    <link rel="stylesheet" href="style.css">
    One stylesheet can style many pages and makes your code organized. [web:1][web:8]

03. Linking an External CSS File

External CSS lives in a separate .css file and is linked with a <link> tag inside the <head>. The rel="stylesheet" and href="style.css" attributes tell the browser where the design file is. [web:5][web:8] 
<!-- index.html -->
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Neon Portfolio</title>

    <!-- Connect external CSS file -->
    <link rel="stylesheet" href="neon-style.css">
</head>
<body>
    <h1>My Neon Cyberpunk Page</h1>
    <p>Styled from an external CSS file!</p>
</body>
</html>
In neon-style.css you put only CSS, no HTML tags. This separation makes your layout easy to reuse and faster to load because the CSS file can be cached. [web:1][web:10] 
/* neon-style.css */
body {
    background: #000000;
    color: #e0e0e0;
    font-family: 'Orbitron', monospace;
}

h1 {
    color: #ff00ff;
    text-align: center;
}

p {
    color: #00ffff;
}

04. Flexbox Superpower Layout

Flexbox is used to create responsive layouts like navbars, card grids, and centered boxes. You turn a container into a flexbox with display: flex; and then control alignment with properties like justify-content and align-items. [web:3][web:6] 
.flex-container {
    display: flex;              /* activate flexbox */
    justify-content: center;    /* horizontal alignment */
    align-items: center;        /* vertical alignment */
}
CENTERED WITH FLEXBOX

05. Flexbox Navbar Example

A navbar is a row of links; Flexbox makes it easy to space the logo on the left and links on the right. justify-content: space-between; pushes items to the edges, while align-items: center keeps them vertically aligned. [web:3][web:6] 
<!-- HTML -->
<nav class="flex-demo-nav">
    <div class="logo">NEON NAV</div>
    <ul>
        <li><a href="#">Home</a></li>
        <li><a href="#">Projects</a></li>
        <li><a href="#">Contact</a></li>
    </ul>
</nav>

/* CSS */
.flex-demo-nav {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

06. Card Grids with Flexbox

A card grid is just a flex container that wraps its children into multiple rows. Using flex-wrap: wrap and gap creates a responsive layout without complex floats. [web:3][web:6] 
<!-- HTML -->
<div class="flex-card-grid">
    <div class="flex-card">
        <h3>Project 1</h3>
        <p>Enrollment System</p>
    </div>
    <div class="flex-card">
        <h3>Project 2</h3>
        <p>Gamified Learning App</p>
    </div>
    <div class="flex-card">
        <h3>Project 3</h3>
        <p>Neon Portfolio</p>
    </div>
</div>

/* CSS */
.flex-card-grid {
    display: flex;
    flex-wrap: wrap;
    gap: 20px;
    justify-content: center;
}
