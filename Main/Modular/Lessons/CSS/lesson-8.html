02. Border Radius

The border-radius property rounds the corners of boxes, cards, and buttons. A high radius (like 50%) can create circular shapes! 
div.card {
  border: 3px solid #ff00ff;
  border-radius: 15px;
  padding: 20px;
}
Rounded Card
Circle!

03. Outlines

Outlines are similar to borders but do not affect layout. They sit on top of the element â€” useful for highlighting focus states. Example: outline: 2px dashed #ff00ff; 
div:focus {
  outline: 3px dashed #ff00ff;
}

04. Borders for Cards & Buttons

Borders and radius are great for designing interactive UI. Combine them for neon-style cards and glowing buttons. 
.neon-card {
  border: 2px solid #00ffff;
  border-radius: 10px;
  background: rgba(0,0,50,0.8);
  padding: 20px;
  box-shadow: 0 0 30px #00ffff99;
}
