The border property outlines elements! Use border: 3px solid #ff0000; for width, style, and color.

Border Radius

border-radius rounds corners! Use pixels or % (50% = circle).
div.card {
  border: 3px solid #ff00ff;
  border-radius: 15px;
  padding: 20px;
}

CSS Outlines

outline = border that doesn't push content! Perfect for focus states and highlights.
button:focus {
  outline: 3px dashed #ff00ff;
}

Cards & Buttons

Combine borders + radius for neon cards and glowing buttons!
.neon-card {
  border: 2px solid #00ffff;
  border-radius: 10px;
  box-shadow: 0 0 20px #00ffff50;
}
