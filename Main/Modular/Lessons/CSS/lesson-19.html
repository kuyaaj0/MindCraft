Shadows, Gradients & Borders

LESSON 19 - Cyberpunk Visual Effects

01. box-shadow

box-shadow: Creates glowing shadows around boxes. Syntax: box-shadow: X Y BLUR RADIUS COLOR. Multiple layers = neon glow! 
.cyber-glow {
  box-shadow: 
    0 0 20px #ff00ff,     /* Pink glow */
    0 0 40px #00ffff,     /* Blue glow */
    0 0 60px rgba(255,0,255,0.5); /* Outer glow */
}
CYBERPUNK BOX GLOW

02. text-shadow

text-shadow: Makes text glow like neon signs! Multiple shadows = realistic cyberpunk effect. Syntax: text-shadow: X Y BLUR COLOR 
h1.neon {
  text-shadow: 
    0 0 10px #ff00ff,
    0 0 20px #00ffff,
    0 0 30px #ff00ff,
    0 0 40px #00ffff;
}

NEON TEXT GLOW

03. linear-gradient()

linear-gradient() creates smooth color transitions. Perfect for cyberpunk backgrounds and borders! Syntax: linear-gradient(angle, color1, color2) 
.gradient-bg {
  background: linear-gradient(45deg,
    #ff00ff 0%,
    #8a2be2 25%,
    #00ffff 50%,
    #ff1493 75%,
    #ff00ff 100%);
  background-size: 400% 400%;
  animation: gradientShift 8s ease infinite;
}

04. border-image + gradient

border-image: Creates gradient borders! Use border-image: linear-gradient(...) 1; for cyberpunk edges. 
.gradient-border {
  border: 4px solid transparent;
  border-image: linear-gradient(45deg,
    #ff00ff, #00ffff, #ff1493) 1;
  border-image-slice: 1;
}
GRADIENT BORDER EFFECT

05. Advanced Combinations

Combine ALL effects for maximum cyberpunk! box-shadow + text-shadow + linear-gradient = ULTIMATE
  • Ultimate Cyber Card:
    box-shadow → Multi-layer neon glow
    border-image → Gradient border
    background → Animated gradient
    text-shadow → Neon text
  • ⚠️ Performance Tip:
    Too many shadows = slow! Use will-change: box-shadow; for smooth animations.
