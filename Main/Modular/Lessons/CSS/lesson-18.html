01. What is transition?

transition creates smooth animations between states (normal â†’ hover). The secret is transition: all 0.4s cubic-bezier(0.25, 0.46, 0.45, 0.94) - this makes everything animate smoothly over 0.4 seconds with a natural easing curve. 
/* BASIC TRANSITION */
.button {
    transition: all 0.4s ease;  /* all properties, 0.4s, smooth easing */
}

.button:hover {
    transform: translateY(-5px);
    box-shadow: 0 10px 30px rgba(0,255,255,0.5);
}

/* SPECIFIC PROPERTIES */
.card {
    transition: transform 0.3s ease, box-shadow 0.3s ease;
}

02. Neon Button Animations

Buttons lift up, glow brighter, and show a shimmer effect on hover. The ::before pseudo-element creates the moving light sweep across the button. 
.cyber-button {
    padding: 18px 35px;
    border: 2px solid rgba(255,0,255,0.5);
    transition: all 0.4s cubic-bezier(0.25, 0.46, 0.45, 0.94);
}

.cyber-button::before {
    content: '';
    position: absolute;
    left: -100%;
    width: 100%;
    height: 100%;
    background: linear-gradient(90deg, transparent, rgba(255,255,255,0.3));
    transition: left 0.6s ease;
}

.cyber-button:hover::before {
    left: 100%;  /* shimmer moves across */
}

.cyber-button:hover {
    transform: translateY(-8px) scale(1.05);
    box-shadow: 0 15px 40px rgba(0,255,255,0.6);
}

03. 3D Card Animations

Cards lift up with 3D rotation and glow effects. The top border shimmers when hovered using a scaling transform on the ::before element. 
.cyber-card {
    border: 2px solid rgba(0,255,255,0.4);
    transition: all 0.4s cubic-bezier(0.25, 0.46, 0.45, 0.94);
}

.cyber-card::before {
    content: '';
    position: absolute;
    top: 0; left: 0; right: 0;
    height: 4px;
    background: linear-gradient(90deg, #ff00ff, #00ffff);
    transform: scaleX(0);
    transition: transform 0.4s ease;
}

.cyber-card:hover {
    transform: translateY(-15px) rotateX(5deg);
    border-color: #ff00ff;
    box-shadow: 0 25px 60px rgba(255,0,255,0.5);
}

.cyber-card:hover::before {
    transform: scaleX(1);  /* border glow appears */
}

05. Best Easing Functions

cubic-bezier(0.25, 0.46, 0.45, 0.94) = smooth natural motion
ease = simple smooth
ease-in-out = slow start + end
Never use linear - feels robotic!
  • transition: all 0.4s cubic-bezier(0.25, 0.46, 0.45, 0.94);
    Perfect for buttons and cards - feels premium!
  • AVOID: transition: all 0s; or no transition
    Instant changes feel cheap and unprofessional.
