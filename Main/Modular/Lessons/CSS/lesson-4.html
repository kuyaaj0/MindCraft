CSS Colors & Backgrounds

Lesson 4 - Mastering Visual Design

01. Why Are Colors and Backgrounds Important?

CSS colors and backgrounds define the look and feel of your webpage. They help create visual contrast, highlight sections, and enhance user experience. You can color text, fill elements with backgrounds, and even use images or gradients.

02. The color Property

The color property changes the text color of an element. You can use:
  • Color names (red, blue)
  • Hex codes (#ff00ff)
  • RGB/RGBA (rgb(255,0,255) or rgba(255,0,255,0.8))
h1 {
    color: #ff00ff;  /* magenta heading */
}
p {
    color: rgba(0, 255, 255, 0.9);  /* neon cyan text */
}

This heading uses magenta color

This paragraph is in neon cyan!

03. The background-color Property

The background-color property fills the background of an element with color. You can apply it to any element, such as div, body, or button. 
div {
    background-color: #111133;
    color: #00ffff;
    padding: 20px;
    border-radius: 10px;
}
This box has a dark background with cyan text.

04. The background-image Property

The background-image property sets an image as a background. It can be an external file or a gradient. 
body {
    background-image: url('neon-grid.png');
    background-repeat: no-repeat;
    background-size: cover;
}

This section uses a gradient background instead of a solid color!
