Uses images or gradients as backgrounds. Perfect for cyberpunk effects!

background-size Property

Lesson 4 â€¢ Topic 5
Controls how background image fits: cover (fills), contain (fits), auto (original).

  background-image: url('pattern.png');
  background-size: cover;  /* fills completely */
}

background-size: cover = full coverage!
