CSS = Cascading Style Sheets. Makes HTML pretty! Controls colors, fonts, spacing, layout. Lives in <style> tags or .css files.
<style>
body {
  background: black;
  color: cyan;
  font-family: monospace;
}
</style>

CSS Selectors

Selectors target HTML elements. Use class (.), ID (#), or element names.
p { color: red; }           /* All paragraphs */
.title { font-size: 24px; } /* Class selector */
#main { background: blue; } /* ID selector */

CSS Properties

Properties = what you style. Common ones: color, background, font-size, margin, padding, border.
color: #ff0000;        /* Red text */
margin: 10px;          /* All sides */
padding: 20px 10px;    /* Top/bottom, left/right */
border: 2px solid blue;

CSS Values

Values = HOW to style. Use pixels (px), percentages (%), colors (hex/rgb/names), keywords (bold/italic).
font-size: 16px;
width: 50%;
color: #00ff00;
background: rgb(0,0,255);
font-weight: bold;

CSS Syntax Rules

Syntax: selector { property: value; }. Use semicolons between properties. Use curly braces.
/* WRONG */
body { color red font big }

/* CORRECT */
body {
  color: red;
  font-size: 24px;
}
