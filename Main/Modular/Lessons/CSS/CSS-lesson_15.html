01. What is CSS Grid?

CSS Grid is a 2D layout system for creating rows and columns at the same time. [web:13][web:14] It is perfect for dashboards, galleries, and multi-column layouts because you can control columns, rows, and gaps between items. [web:13][web:17] 
.grid-container {
    display: grid;              /* Turn this element into a grid container */
    grid-template-columns: 1fr 1fr 1fr;  /* 3 equal-width columns */
    gap: 20px;                  /* Space between rows and columns */
}

02. grid-template-columns

grid-template-columns defines how many columns you have and how wide each column is. [web:12][web:13] You can use fixed units (px), percentages, or flexible units like fr (fraction of free space). [web:14] 
.two-columns {
    display: grid;
    grid-template-columns: 1fr 1fr;   /* two equal columns */
}

.three-columns {
    display: grid;
    grid-template-columns: 200px 1fr 1fr; /* first is fixed 200px, others share remaining space */
}

.auto-layout {
    display: grid;
    grid-template-columns: repeat(3, 1fr); /* 3 equal columns using repeat() */  /* [web:13] */
}

03. grid-template-rows

grid-template-rows sets the height of each row in the grid. [web:13][web:14] If you do not set it, rows automatically grow to fit their content, which is often fine for basic layouts. [web:14] 
.fixed-rows {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: 80px 120px; /* first row 80px, second row 120px */
    gap: 10px;
}

04. gap (row & column gap)

The gap property sets the empty space between rows and columns, like gutters between cards. [web:17][web:20] You can give one value for both directions, or two values: first for row-gap and second for column-gap. [web:14][web:20] 
.equal-gap {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 20px;           /* 20px between rows and columns */
}

.different-gap {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px 40px;      /* 10px row-gap, 40px column-gap */
}
  • Clean card layout:
    Use gap: 20px; to avoid elements sticking to each other and improve readability in dashboards and galleries. [web:17][web:20]
  • Do not use margins everywhere:
    Using margin on each card can be harder to control; gap is made specifically for grid spacing. [web:17]

05. Simple Two-Column Layout

A two-column layout is common for sidebar + main content pages. [web:12][web:18] You can give the sidebar less space and the main content more space using fractional units like 1fr 3fr. [web:18] 
<!-- HTML -->
<div class="layout-two-column">
    <aside class="sidebar">Sidebar (links, menu)</aside>
    <main class="main-content">Main content area</main>
</div>

/* CSS */
.layout-two-column {
    display: grid;
    grid-template-columns: 1fr 3fr;  /* sidebar smaller, main bigger */
    gap: 20px;
}

06. Simple Three-Column Layout

A three-column layout is useful for cards, features, or stats in a row. [web:13][web:16] Using grid-template-columns: repeat(3, 1fr); creates three equal columns that adapt to the container. [web:13][web:14] 
<!-- HTML -->
<section class="features-grid">
    <div class="feature">Column 1</div>
    <div class="feature">Column 2</div>
    <div class="feature">Column 3</div>
</section>

/* CSS */
.features-grid {
    display: grid;
    grid-template-columns: repeat(3, 1fr);  /* 3 equal columns */
    gap: 20px;
}

07. YOUR CHALLENGE!

Copy the starter template below and build your own CSS Grid page with 2-column and 3-column sections. Customize it for your portfolio, school subjects, or IT projects. 
<!DOCTYPE html>

            gap: 20px;

        .three-cols {
            display: grid;
            grid-template-columns: repeat(3, 1fr);
            gap: 15px;
        }

            border: 2px solid #ff00ff;
            padding: 15px;
            border-radius: 10px;
        }
    </style>
</head>

            </div>
