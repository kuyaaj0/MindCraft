Script Tags

Use <script> tags in HEAD or BODY or External JS.
<script defer>console.log("HEAD loaded")</script>
<script>alert("BODY script")</script>
<script src="script.js"></script>

External JS Files

Put JS in a separate file for better management.
<script src="app.js"></script>
console.log("External JS loaded");
document.getElementById("demoExt").innerHTML="Loaded";
External JS not loaded

Console

Use console.log(), console.warn(), console.error() for debugging.
console.log("Info");
console.warn("Warning");
console.error("Error");
// Console output here

Alert Popups

Use alert() to show popup messages to users.
alert('Hello!');
alert('Score: 100');
alert('Game Over!');

innerHTML

Change HTML content dynamically using innerHTML.
<div id="demoDiv">Loading...</div>
document.getElementById("demoDiv").innerHTML="Hello Pixel!";
Loading...
