02. push() & pop()

The push() method adds a new item to the end of the array. The pop() method removes the last item. 
let cyberGears = ["Helmet", "Gloves", "Boots"];
cyberGears.push("Visor");
console.log(cyberGears); 

cyberGears.pop();
console.log(cyberGears);
Output â†’ ["Helmet", "Gloves", "Boots", "Visor"]

03. shift()

The shift() method removes the first element of an array and shifts all others left. 
let players = ["V", "Johnny", "Panam"];
players.shift();
console.log(players);
Output â†’ ["Johnny", "Panam"]

04. map()

The map() method creates a new array by performing a function on every element. 
let digits = [1, 2, 3, 4];
let doubleDigits = digits.map(num => num * 2);
console.log(doubleDigits);
Output â†’ [2, 4, 6, 8]

05. filter()

The filter() method returns a new array with elements that pass a test condition. 
let levels = [10, 25, 40, 55, 70];
let high = levels.filter(level => level > 40);
console.log(high);
Output â†’ [55, 70]

07. find()

The find() method returns the first element that satisfies a condition.

08. sort()

The sort() method arranges array items alphabetically or numerically (if compared properly). 
let cities = ["Tokyo", "Manila", "Seoul", "Osaka"];
cities.sort();
console.log(cities);

let numbers = [30, 5, 100, 2];
numbers.sort((a, b) => a - b);
console.log(numbers);
