01. Object Literals

An object stores multiple related values as key-value pairs. Think of it like a futuristic data container: each ‚Äúproperty‚Äù has a name and a value. 
let player = {
  name: "V",
  level: 45,
  weapon: "Katana",
  alive: true
};

console.log(player);
Output ‚Üí { name: "V", level: 45, weapon: "Katana", alive: true }

02. Property Access

You can access object data using two ways: dot notation (player.name) and bracket notation (player["weapon"]). 
console.log(player.name);     // V
console.log(player["weapon"]); // Katana
Output ‚Üí V
Katana

    return `üöÄ Boosting to ${this.speed + 50} km/h!`;

05. Basic Object Patterns

Objects are widely used ‚Äî for **user profiles, settings, game data, and APIs**. You can combine them with arrays or functions for powerful designs.

  { item: "Pistol", ammo: 30 },
  { item: "Medkit", amount: 2 },
  { item: "Cyberdeck", upgrade: true }
];

inventory.forEach(obj => console.log(`Item: ${obj.item}`));
Output ‚Üí
Item: Pistol
Item: Medkit
