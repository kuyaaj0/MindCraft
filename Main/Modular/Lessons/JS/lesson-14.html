Object Literals

Lesson 14 ‚Ä¢ Topic 1
Objects store data in key-value pairs using {} braces.
Perfect for game characters, enemies, items, and game state! 
let player = {
  name: "PixelHero",
  level: 5,
  hp: 100,
  coins: 50
};

console.log(player);
{name: 'PixelHero', level: 5, hp: 100, coins: 50}
// Objects group related data!

Property Access

Lesson 14 ‚Ä¢ Topic 2
Get properties with dot notation object.property or
bracket notation object["property"]. 
let enemy = {
  name: "DarkBot",
  hp: 80,
  attack: 15
};

// Dot notation (most common)
console.log(enemy.name);     // "DarkBot"

// Bracket notation
console.log(enemy["hp"]);    // 80

// Update values
enemy.hp -= 20;              // Take damage

Nested Objects

Lesson 14 ‚Ä¢ Topic 3
Objects can contain other objects inside properties.
Chain dots to access: object.nested.property. 
let hero = {
  name: "Nova",
  stats: {
    hp: 120,
    attack: 25,
    defense: 15
  },
  weapon: {
    name: "Laser Sword",
    damage: 30
  }
};

console.log(hero.stats.hp);
console.log(hero.weapon.name);
120
Laser Sword

Common Patterns

Lesson 14 ‚Ä¢ Topic 4
Objects can hold functions (methods) too!
Use this keyword inside methods to reference the object. 
let goblin = {
  name: "Goblin",
  hp: 50,
  attack: function() {
    console.log(this.name + " attacks for " + this.attack + "!");
  }
};

goblin.attack();

Goblin attacks for 15!
