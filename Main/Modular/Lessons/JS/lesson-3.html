02. Multi-Line Comments /* */

/* */ = Multiple lines comment. Everything between /* and */ = ignored. Use for function descriptions, long explanations, temporarily disabled code. 
/* ========================================
   CYBERPUNK STUDENT GRADE CALCULATOR v2.0
   Created by: Jhack @ CMU
   Purpose: Calculate final grades with bonus
========================================= */

/* 
   FUNCTION: calculateGrade
   INPUT: current grade (0-100)
   OUTPUT: final grade with bonus
   RULES: A=90+, B=80-89, C=70-79
*/
function calculateGrade(score) {
    if (score >= 90) {
        return "A - ELITE HACKER!";
    }
    return "Keep hacking!";
}

03. GOOD vs BAD Comments

GOOD comments EXPLAIN WHY, not what. BAD comments repeat obvious code. Future you (6 months later) will thank you for good comments!
  • ‚úÖ GOOD COMMENTS (Explain WHY):
    let bonus = 5; // Extra points for perfect attendance
    total -= fee; // Remove enrollment fee before final calc

  • ‚ùå BAD COMMENTS (Obvious):

    if (x > 0) {} // Check if positive ‚ùå

04. JSDoc Style Comments

JSDoc = Professional documentation format. Starts with /** ends with */. Used by VS Code for auto-completion and documentation. 
/**
 * Calculate student final grade
 * @param {number} score - Current grade (0-100)
 * @param {boolean} attendedAll - Perfect attendance?
 * @returns {string} Grade letter (A,B,C,D,F)
 */
function getFinalGrade(score, attendedAll) {
    let finalScore = score;
    
    if (attendedAll) {
        finalScore += 5;  // Attendance bonus
    }
    
    if (finalScore >= 90) return "A";
    if (finalScore >= 80) return "B";
    return "C";
}
