Map Basics

Lesson 15 â€¢ Topic 1
Map stores key-value pairs like objects, but
keys can be ANY type (not just strings) and keeps insertion order! 
let playerInventory = new Map();

// Add items (any key type!)
playerInventory.set("sword", 1);
playerInventory.set(42, "health potion");
playerInventory.set(true, "shield");

console.log(playerInventory.get("sword"));
1
// Map.get() retrieves values!

Set Basics

Lesson 15 â€¢ Topic 2
Set stores UNIQUE values only - no duplicates!
Perfect for tracking collected items, visited levels, or unique enemies. 
let collectedGems = new Set();

collectedGems.add("red");
collectedGems.add("blue");
collectedGems.add("red");  // Ignored - duplicate!

console.log(collectedGems.size);  // 2 (unique only!)
2
// Sets automatically remove duplicates!

typeof Operator

Lesson 15 â€¢ Topic 3
typeof tells you what data type a value is.
