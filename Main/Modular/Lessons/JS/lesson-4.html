Naming Rules

Names must start with letter, _, or $.

Reassignment

Lesson 4 ‚Ä¢ Topic 4
let variables can get new values anytime.
const stays the same forever. 
let level = 1;
console.log("Start:", level);

level = 2;               // Change it!
console.log("Now:", level);

level += 1;              // Add 1
console.log("Final:", level);
Start: 1
Now: 2
Final: 3
