02. Script Tags (3 Locations)

JavaScript lives inside <script> tags. Place them in 3 locations: HEAD (use defer), BODY END (best), or External .js. 
<!-- METHOD 1: HEAD (use defer) -->
<head>
<script defer>
    console.log("Loading...");
</script>
</head>

<!-- METHOD 2: BODY END (BEST) -->
<body>
    <h1>Content</h1>
    <script>
        alert("Page loaded!");
    </script>
</body>

<!-- METHOD 3: EXTERNAL -->
<script src="cyberpunk.js"></script>

‚úÖ All 3 methods work perfectly!
