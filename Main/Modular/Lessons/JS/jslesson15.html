Maps, Sets & Type Conversion

Lesson 15 - Cyberpunk Data Systems

01. Map Basics

A Map stores key-value pairs (like an object) — but keys can be of any type (even objects!). Use methods like set(), get(), and has() to work with data. 
let hackerMap = new Map();
hackerMap.set("alias", "Neo");
hackerMap.set("level", 9000);
hackerMap.set("gear", ["Gloves", "Visor"]);

console.log(hackerMap.get("alias"));
console.log(hackerMap.has("level"));
console.log(hackerMap.size);
Output → Neo
true
3

02. Set Basics

A Set stores unique values — duplicates are automatically removed. Ideal for lists where repeat entries aren’t allowed. 
let neonSet = new Set(["cyber", "punk", "cyber"]);
neonSet.add("matrix");
console.log(neonSet);
console.log(neonSet.size);
Output → {"cyber", "punk", "matrix"}
3

03. typeof Operator

The typeof operator identifies the data type of a value or variable. Perfect for debugging type mismatches. 
let name = "V";
let age = 27;
let active = true;
let tools = ["deck", "blade"];
console.log(typeof name);
console.log(typeof age);
console.log(typeof active);
console.log(typeof tools);
Output → string
number
boolean
object

04. Type Conversion

You can change data types manually using constructors: String(), Number(), and Boolean(). 
let score = "100";
let realNumber = Number(score);  
console.log(realNumber + 20);

let active = false;
let text = String(active);
console.log("Converted:", text);

let value = 0;
console.log(Boolean(value)); // false
Output → 120
Converted: false
false

05. Combining Maps, Sets, and Conversion

You can use all three features together for professional data handling — e.g., converting a Set to an Array from a Map. 
let cyberMap = new Map([
  ["city", "Night City"],
  ["fame", 99]
]);
let cyberSet = new Set(["Neon", "Chrome", "Neon"]);

let arrayFromSet = Array.from(cyberSet);
let mapKeys = Array.from(cyberMap.keys());

console.log(arrayFromSet);
console.log(mapKeys);
Output → ["Neon", "Chrome"]
["city", "fame"]
