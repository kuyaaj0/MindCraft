01. String Length

The length property counts the number of characters (letters, spaces, symbols) in a string. 
let text = "Cyberpunk";
let count = text.length;
console.log(count); // 9
Output â†’ 9

02. slice()

The slice(start, end) method extracts part of a string and returns it as a new one. The end index is not included. 
let word = "Neon Future";
let result = word.slice(0, 4);
console.log(result); // Neon
Output â†’ Neon

03. substring()

The substring(start, end) works similar to slice but does not accept negative indexes. 
let name = "Cyber Universe";
let part = name.substring(6, 14);
console.log(part); // Universe
Output â†’ Universe

04. replace()

The replace() method swaps a part of a string with another text. Only the first match is replaced by default. 
let phrase = "I love Java.";
let newPhrase = phrase.replace("Java", "JavaScript");
console.log(newPhrase); // I love JavaScript.
Output â†’ I love JavaScript.

05. split()

The split() method breaks a string into an array using a separator (like a space or comma). 
let sentence = "Code Neon Dreams";
let words = sentence.split(" ");
console.log(words); // ["Code", "Neon", "Dreams"]
Output â†’ ["Code", "Neon", "Dreams"]

06. Template Literals

Template literals use backticks (`) to allow embedding variables easily using ${}. 
let hero = "V";
let city = "Night City";
let message = `Hello, ${hero}! Welcome to ${city}.`;
console.log(message);
// Hello, V! Welcome to Night City.
Output â†’ Hello, V! Welcome to Night City.
