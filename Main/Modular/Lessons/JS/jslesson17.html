01. What Are Events?

Events are actions that happen in the browser â€” like clicking, typing, submitting a form, or hovering over an element. JavaScript can respond to these dynamically to make websites interactive.

02. The onclick Event

The simplest event in JavaScript is onclick. It runs code when the user clicks the element. 
<button onclick="alert('Cyber click!')">Click Me</button>
Output â†’ A pop-up alert appears when the button is clicked.

03. addEventListener()

Instead of inline code, we often use addEventListener() â€” it "listens" for actions and runs a function when an event occurs. 
<button id="hackerBtn">Activate</button>

<script>
let button = document.getElementById("hackerBtn");
button.addEventListener("click", function(){
  alert("Access Granted!");
});
</script>
Output â†’ Alert "Access Granted!" appears when pressed.

04. Handling Form Submissions

Forms can trigger submit events. Use preventDefault() to stop reloading the page. 
<form id="loginForm">
  <input type="text" placeholder="Username">
  <button>Login</button>
</form>

<script>
let form = document.getElementById("loginForm");
form.addEventListener("submit", function(event){
  event.preventDefault(); // Stop page refresh
  alert("Form submitted successfully!");
});
</script>
Output â†’ Alert displays without refreshing the page.

05. Key Press Events

The keydown event fires when any key is pressed on the keyboard. Perfect for game controls or dynamic text effects! 
<input type="text" id="inputBox" placeholder="Type something...">

<script>
let box = document.getElementById("inputBox");
box.addEventListener("keydown", function(e){
  console.log("Key pressed:", e.key);
});
</script>
Output â†’ Logs each key you press inside the input box.

06. Mouse Events

You can also capture hovering or leaving an element using mouseover and mouseout. 
<div id="panel">Hover over me!</div>

<script>
let panel = document.getElementById("panel");

panel.addEventListener("mouseover", () => {
  panel.style.color = "#00ffff";
  panel.textContent = "Neon Energy Detected!";
});

panel.addEventListener("mouseout", () => {
  panel.style.color = "#ff00ff";
  panel.textContent = "Hover again!";
});
</script>
Output â†’ The text and color change when you hover and leave.
