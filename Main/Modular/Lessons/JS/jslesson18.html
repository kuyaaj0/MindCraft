01. What is DOM Manipulation?

DOM manipulation means changing your webpage structure and content using JavaScript. You can create new elements, delete them, or modify their styles and classes dynamically.

02. Creating Elements

Use document.createElement() and appendChild() to add brand-new elements to your HTML. 
<div id="container"></div>

<script>
let box = document.createElement("p");
box.textContent = "Welcome to the Neon Grid!";
document.getElementById("container").appendChild(box);
</script>
Output ‚Üí A paragraph appears saying ‚ÄúWelcome to the Neon Grid!‚Äù

03. Removing Elements

You can remove elements using remove() or removeChild() methods. 
<div id="arena">
  <p>Cyber Element 1</p>
  <p id="removeMe">Remove This One</p>
</div>

<script>
let target = document.getElementById("removeMe");
target.remove(); // Instantly deletes the element
</script>
Output ‚Üí ‚ÄúRemove This One‚Äù disappears instantly.

04. Changing Classes & Styles

The classList property allows adding, removing, and toggling CSS classes dynamically. This makes interactive styling easy and effective. 
<style>
.glow {
  color: #00ffff;
  text-shadow: 0 0 20px #00ffff;
}
</style>

<h3 id="neonText">Activate Glow</h3>
<button id="toggleBtn">Toggle Glow</button>

<script>
let neon = document.getElementById("neonText");
let btn = document.getElementById("toggleBtn");

btn.addEventListener("click", () => {
  neon.classList.toggle("glow");
});
</script>
Output ‚Üí Click toggles glowing neon effect on text.

05. Updating Inline Styles

The style property lets you directly control element CSS through JavaScript. You can modify colors, positions, and animations in real-time. 
<div id="cyberBox">Neural Block</div>
<button id="moveBtn">Shift Position</button>

<script>
let box = document.getElementById("cyberBox");
let move = document.getElementById("moveBtn");

move.addEventListener("click", () => {
  box.style.color = "#ff00ff";
  box.style.transform = "translateX(100px)";
  box.style.transition = "0.8s";
});
</script>
Output ‚Üí ‚ÄúNeural Block‚Äù moves sideways and changes to pink.

06. Cyber Building Challenge

Try creating an app that adds new glowing messages every time you click a button, and removes one message with another button. Style them dynamically for extra glow. 
<div id="messageArea"></div>
<button id="addMsg">Add Message</button>
<button id="removeMsg">Remove Last</button>

<script>
let area = document.getElementById("messageArea");
let add = document.getElementById("addMsg");
let remove = document.getElementById("removeMsg");

add.addEventListener("click", () => {
  let msg = document.createElement("p");
  msg.textContent = "‚ö° Neon message added!";
  msg.style.color = "#00ffff";
  msg.style.textShadow = "0 0 20px #00ffff";
  area.appendChild(msg);
});

remove.addEventListener("click", () => {
  if (area.lastChild) area.lastChild.remove();
});
</script>
Output ‚Üí Adds or removes glowing messages dynamically.
