01. Semantic HTML5 Tags

                

                    header, nav, main, section, article, footer = 
                    tell browsers "what" content means, not just "how" it looks.
                

                

                    
Top branding/navigation

                    
Main menu links

                    
Primary page content

                    
Themed content blocks

                    
Independent content

                    
Page bottom info

                

            


            

                
02. Perfect Page Structure

                

                    Every page needs: header → nav → main → footer. 
                    Use section and article inside main.
                

                

                    
<header>Logo + Title</header>
<nav>Menu links</nav>
<main>
    <section>Content area 1</section>
    <article>Blog post</article>
</main>
<footer>Copyright</footer>

                

            


            

                
03. Accessibility Features

                

                    alt="" = image descriptions for screen readers. 
                    label for="" = connects labels to inputs.
                

                

                    
                

            


            

                
04. Accessible Forms

                

                    Always use label for="id" matching input id.
