02. Absolute vs Relative URLs

ABSOLUTE = Full URL (https://...)
RELATIVE = File path (folder/file.html)
  • ABSOLUTE: <a href="https://facebook.com">FB</a>
    Works from anywhere!
  • RELATIVE: <a href="images/photo.jpg">Photo</a>
    Same folder structure needed

03. target="_blank"

target="_blank" = Opens link in NEW tab
No target = Same tab (default) 
<a href="https://youtube.com" target="_blank">YouTube (NEW TAB)</a>
<a href="https://w3schools.com">W3Schools (SAME TAB)</a>
üé• YouTube (NEW TAB) üìö W3Schools (SAME TAB)

04. Special Links

mailto: = Email links
tel: = Phone links

    Opens your email app!
  • <a href="tel:+639123456789">Call</a>
    Opens phone dialer
  • <a href="resume.pdf" download>Download</a>
