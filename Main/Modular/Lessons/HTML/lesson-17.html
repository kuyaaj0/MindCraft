Numbers

Lesson 1
In JavaScript, numbers are used for various calculations and operations.
let x = 5;
let y = 10;
let result = x + y;
console.log(result);

Date

You can work with dates and times in JavaScript using the Date object.
let date = new Date();
console.log(date);

Range

The range of values can be manipulated with loops and functions like Math.random().
let randomNum = Math.floor(Math.random() * 100);
console.log(randomNum);

File

JavaScript can interact with files on the client side using File APIs.
<input type="file" id="fileInput" />
document.getElementById('fileInput').addEventListener('change', function(event){
  console.log(event.target.files[0].name);
});
