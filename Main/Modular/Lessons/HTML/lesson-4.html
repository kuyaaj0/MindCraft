Absolute/Relative URLs

Lesson 4 ‚Ä¢ Topic 1/6
Absolute: Full URLs like https://google.com
Relative: Site paths like about.html or images/logo.png
Use relative for same website, absolute for external sites! 
<!-- ABSOLUTE URLs -->
<a href="https://google.com">Google</a>
<a href="https://github.com">GitHub</a>

<!-- RELATIVE URLs -->
<a href="about.html">About</a>
<a href="images/logo.png">Logo</a>
<a href="level1.html">Level 1</a>
