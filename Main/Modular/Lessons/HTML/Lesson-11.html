Using CSS with HTML

Lesson 11 - Cyberpunk Styling Matrix

01. What is CSS?

CSS, or Cascading Style Sheets, defines how HTML elements look — like their color, size, layout, or animation. HTML structures your page; CSS gives it style and personality.

02. Inline CSS

Inline CSS is written directly inside an HTML tag using the style attribute. It applies only to that element. 
<h2 style="color:deeppink; font-size:30px;">Hello Neon!</h2>

Hello Neon!

03. Internal CSS

Internal CSS is placed inside the <style> tag in the <head> section. It styles elements across the same page. 
<style>
    p {
        color: cyan;
        font-family: 'Roboto Mono', monospace;
    }
</style>

<p>This is styled using internal CSS.</p>

This is styled using internal CSS.

04. External CSS

External CSS is written in a separate file (like style.css), then linked to your HTML using the <link> tag. This method is best for large websites since one file controls multiple pages. 
<!-- HTML File (index.html) -->
<head>
    <link rel="stylesheet" href="style.css">
</head>

<body>
    <h2>Cyber CSS Demo</h2>
</body>

--- style.css ---
h2 {
    color: #00ffff;
    text-shadow: 0 0 10px #00ffff;
}

Cyber CSS Demo

05. Comparison Summary

Here's how all three types of CSS compare: 
INLINE    → style="color:blue;" (1 HTML element)
INTERNAL  → uses <style> inside <head> (1 HTML page)
EXTERNAL  → linked .css file (multiple pages)
