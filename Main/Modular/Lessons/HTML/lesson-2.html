01. Heading Levels h1 ‚Üí h6

Headings create document structure. h1 = Main Title (use ONLY ONCE per page). h2-h6 = subsections. 
<h1>Main Page Title</h1>
<h2>Section Title</h2>
<h3>Subsection</h3>
<h4>Detail</h4>
<h5>Fine detail</h5>
<h6>Tiniest</h6>

02. Paragraphs <p>

<p> creates text blocks with proper spacing. 
<p>This is a paragraph.</p>
<p>Another paragraph.</p>

03. Line Breaks <br>

<br> forces new line WITHOUT paragraph spacing. 
<p>Line 1<br>Line 2<br>Line 3</p>
