01. Heading Levels h1 ‚Üí h6

Headings create document structure like chapters in a book. h1 = Main Title (use ONLY ONCE per page). h2-h6 = subsections. Size decreases, importance decreases.

02. Paragraphs <p>

<p> creates text blocks with proper spacing. Browsers add margins automatically. Perfect for readable content. 
<p>This is a paragraph. Browsers add space above/below automatically.</p>
<p>Another paragraph. Perfect spacing between paragraphs!</p>

03. Line Breaks <br>

<br> forces new line WITHOUT paragraph spacing. Use for addresses, poems, short breaks. Self-closing tag.

04. Proper Document Structure

Use headings like an outline: h1 (main topic) ‚Üí h2 (sections) ‚Üí h3 (subsections). Paragraphs explain each section.
  • ‚úÖ GOOD STRUCTURE:
    <h1>My Website</h1>
    <h2>About Me</h2>
    <p>My bio...</p>
    <h2>Projects</h2>
  • ‚ùå BAD (jumping levels):
    <h1>Title</h1>
    <h3>Skip h2!</h3> ‚Üê Confusing!

05. SEO & Accessibility Rules

Search engines and screen readers follow your heading order. Never skip levels (h1 ‚Üí h2 ‚Üí h3). Use for structure, not styling.
  • h1 = ONE per page (main topic)
  • h2 = main sections
  • Paragraphs explain each section
  • <br> = line breaks only (poems, addresses)
