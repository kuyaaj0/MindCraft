Colors

Lesson 1
Learn how to use colors in your webpage: color names, hex codes, and RGB. 
document.body.style.color = "red"; // color names
document.body.style.color = "#00ff00"; // hex codes
document.body.style.color = "rgb(0, 0, 255)"; // RGB

Background Colors

Change the background color dynamically. 
document.body.style.backgroundColor = "yellow";
document.body.style.backgroundColor = "#f0f0f0";
document.body.style.backgroundColor = "rgb(200,100,50)";

Basic Theme

Combine text color and background for a simple page theme. 
document.body.style.color = "white";
document.body.style.backgroundColor = "black";

Theme Switcher

Interactive lesson: switch between light, dark, and colorful themes. 
function setTheme(bg, color){
  document.body.style.backgroundColor = bg;
  document.body.style.color = color;
}
