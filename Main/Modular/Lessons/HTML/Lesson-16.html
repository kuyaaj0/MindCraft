LESSON 16 - Cyberpunk Input Matrix

01. Text, Email, Password

Basic inputs: text (normal), email (email validation), password (hidden dots).

02. Textarea & Select

textarea = multi-line text. select = dropdown menu.

03. Checkbox & Radio

checkbox = multiple choices OK. radio = only ONE choice.

04. Button Types

button = custom action. submit = sends form. reset = clears form. 
<button type="button">Custom Action</button>
<button type="submit">Send Data</button>
<button type="reset">Clear Form</button>
