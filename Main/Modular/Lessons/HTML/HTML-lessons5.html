01. Basic Images <img>

<img src="image.jpg" alt="description">
src = file path/URL
alt = text for screen readers (CRITICAL!) 
<img src="photo.jpg" alt="My profile photo">
<img src="https://via.placeholder.com/300" alt="Placeholder image">

‚Üê Neon cyberpunk image with alt text!

02. Size Control (width/height)

Use width and height attributes
IMPORTANT: Always set BOTH for fast loading 
<!-- Small thumbnail -->
<img src="logo.png" width="100" height="100" alt="Logo">

<!-- Full width -->
<img src="banner.jpg" width="800" height="300" alt="Banner">

03. Alt Text (CRITICAL!)

alt="" = What screen readers say
Empty alt="" for decorative images
NO alt = BROKEN for accessibility!
  • ‚úÖ <img src="profile.jpg" alt="Juan Dela Cruz, IT student">
  • ‚ùå <img src="decor.jpg" alt=""> (decorative)
  • ‚ùå <img src="photo.jpg"> (NO alt = BAD!)

04. Responsive Images

width="100%" = Scales to container
CSS max-width:100%; height:auto; = Perfect responsive 
<img src="hero.jpg" width="100%" alt="Hero image">
<!-- CSS makes it responsive -->
img { max-width: 100%; height: auto; }

‚Üê Shrinks on mobile automatically!

05. Image File Paths

Same folder: "photo.jpg"
Subfolder: "images/photo.jpg"
Parent folder: "../photo.jpg"
  • <img src="logo.png" alt="Logo"> ‚Üê Same folder
  • <img src="images/banner.jpg" alt="Banner"> ‚Üê images/ folder
  • <img src="../assets/photo.jpg" alt="Photo"> ‚Üê Go up 1 folder
