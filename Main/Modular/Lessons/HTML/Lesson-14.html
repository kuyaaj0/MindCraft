Responsive Design Basics

LESSON 14 - Cyberpunk Mobile Matrix

01. Viewport Meta Tag

The viewport meta tag tells mobile browsers to show your site correctly. Without it, phones zoom out too much. 
<meta name="viewport" 
    content="width=device-width, initial-scale=1.0">
Desktop: Full width, normal zoom
Mobile: Fits phone screen perfectly (no tiny zoom-out)

02. Percentage Widths

Use % instead of fixed pixels. Elements shrink/grow with screen size automatically. 
.container { width: 90%; max-width: 1200px; }
.box { width: 50%; }  /* Half screen */
.image { width: 100%; }
FIXED 400px - Never changes
50% WIDTH - Shrinks on mobile

03. Responsive Images

Images with width: 100%; height: auto; scale perfectly on all devices without distortion. 
img {
    width: 100%;
    height: auto;
    max-width: 600px;
}

Above image scales perfectly on phones!

04. Media Queries

Change styles based on screen size using @media.
  • Tablets (768px and down):
    @media (max-width: 768px) { h1 { font-size: 2em; } }
  • Phones (480px and down):
    @media (max-width: 480px) { .container { padding: 10px; } }

05. YOUR CHALLENGE!

Make your portfolio responsive! Test on phone browser and resize desktop window.
