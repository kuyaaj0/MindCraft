Text Formatting

Lesson 3
HTML formatting tags make text BOLD, ITALIC, small, add sub/superscripts like H2O, and handle quotes or code snippets perfectly!

  <p>Chemical: H<sub>2</sub>O</p>
  <p>Math: x<sup>2</sup> + 2x</p>
</body>
</html>

Bold & Italic

<strong> shows importance (renders bold), <em> shows emphasis (renders italic). Perfect for pixel games and warnings! 
<!DOCTYPE html>
<html>
<body style="background:#000;color:#0f0;font-family:monospace;">
  <h1><strong>PIXEL QUEST</strong></h1>
  <p><strong>WARNING:</strong> Save often!</p>
  <p>I <em>love</em> making games!</p>
  <p>Next level: <em>very hard</em></p>
</body>
</html>
PIXEL QUEST (BOLD)
WARNING: Save often! (BOLD)
I love making games! (ITALIC)
Next level: very hard (ITALIC)

Sub/Sup Scripts

<sub> drops text below baseline, <sup> raises it above - essential for math, chemistry, and game stats! 
<!DOCTYPE html>
<html>
<body>
  <p>Water: H<sub>2</sub>O</p>
  <p>Carbon: CO<sub>2</sub></p>
  <p>Square: x<sup>2</sup></p>
  <p>Power: 10<sup>3</sup> = 1000</p>
  <p>Game score: Level<sup>5</sup></p>
</body>
</html>

Quotes & Code

<q> adds smart quotes automatically, <code> shows monospace code text - perfect for tutorials and game instructions!
