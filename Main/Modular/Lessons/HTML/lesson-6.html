05. List Best Practices

Lists improve readability and SEO. Use semantic HTML for screen readers.
  • <ul> = No order (shopping, features)
  • <ol> = Order matters (steps, rankings)
  • Never use images as bullets ‚Üí use CSS
  • <nav> = Only main navigation links
