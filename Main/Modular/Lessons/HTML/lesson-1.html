HTML Structure

Lesson 1
HTML (HyperText Markup Language) gives structure to a web page. Every HTML file starts with a doctype, followed by html, head, and body tags. 
<!DOCTYPE html>
<html>
  <head>
    <title>My First Webpage</title>
  </head>
  <body>
    <h1>Hello World!</h1>
  </body>
</html>

Head & Body

Lesson 1
The head contains invisible info (title, CSS links, meta). The body shows visible content only.

Tags & Nesting

Lesson 1
HTML uses opening and closing tags. Tags can be nested inside each other, but they must close in the correct order. 
<div>
  <p>This is <strong>nested</strong> text.</p>
</div>
