HTML Structure

Lesson 1 of 4
HTML (HyperText Markup Language) gives structure to a web page. Every HTML file starts with a doctype, followed by html, head, and body tags. This is the basic skeleton every webpage needs. 
<!DOCTYPE html>
<html>
  <head>
    <title>My First Webpage</title>
  </head>
  <body>
    <h1>Hello World!</h1>
  </body>
</html>

Head & Body

Lesson 2 of 4
The head contains invisible info like the page title (shows in browser tab) and CSS links. The body shows all visible content only. Head works behind the scenes, body is what users see. 
<!DOCTYPE html>
<html>
  <head>
    <title>My Webpage</title>  ‚Üê Tab title only
    <link rel="stylesheet" href="style.css"> ‚Üê Styles
  </head>
  <body>
    <h1>Welcome!</h1>  ‚Üê Visible!
    <p>This shows up.</p>  ‚Üê Visible!
  </body>
</html>

Tags & Nesting

Lesson 3 of 4
HTML uses opening <tag> and closing </tag> tags. Tags can be nested inside each other, but they must close in the correct order (like opening/closing brackets). 
<div>
  <p>This is <strong>nested</strong> text.</p>
</div>
