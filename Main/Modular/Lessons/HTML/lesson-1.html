HTML Structure & Elements

Neon Cyberpunk Edition - Complete Beginner Lesson

1. DOCTYPE Declaration

The <!DOCTYPE html> must be the very first line. It tells browsers this is HTML5.

<!DOCTYPE html>

2. <html> Root Element

The <html> tag is the root container for all content.

<html lang="en">
  <!-- All content here -->
</html>

3. <head> Element

<head> holds metadata only: title, charset, viewport, CSS/JS links.

<head>
  <title>My Page</title>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>

4. <body> Element

<body> contains all visible content: text, images, links, etc.

5. HTML Tags & Elements

Tags create elements with an opening tag, content, and closing tag.

<p>Hello World!</p>
  • Opening Tag: <p>
  • Closing Tag: </p>
  • Content: The text or other elements between the tags.
