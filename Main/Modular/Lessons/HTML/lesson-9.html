01. üéØ Classes (class="name")

Classes let you style MULTIPLE elements with the same CSS rules. Use the class="" attribute. Target with .classname in CSS. Perfect for buttons, cards, navigation items - anything reusable!

<div class="cyber-card">Card 2</div>
<button class="neon-btn">Click Me</button>
<button class="neon-btn">Click Me Too</button>

02. üÜî IDs (id="unique")

IDs target ONE UNIQUE element per page. Use id="" attribute. Target with #idname in CSS. Perfect for hero sections, main content, unique navigation bars. 
<!-- HTML (ONLY ONE per page!) -->
<h1 id="hero-title">Welcome to Cyberpunk</h1>
<nav id="main-menu">Navigation</nav>

<!-- CSS -->
