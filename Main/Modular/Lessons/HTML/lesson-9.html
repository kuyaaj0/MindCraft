Script Tags

Use <script> tags in HEAD or BODY or External JS.
<script defer>console.log("HEAD loaded")</script>
<script>alert("BODY script")</script>
<script src="script.js"></script>

External JS Files

Put JS in a separate file for better management.
<script src="app.js"></script>
console.log("External JS loaded");
document.getElementById("demoExt").innerHTML="Loaded";

Console

Use console.log(), console.warn(), console.error() for debugging.
console.log("Info");
console.warn("Warning");
console.error("Error");
// Console output here

Functions

Functions let you reuse JS code efficiently.
function greet(name){
  console.log("Hello " + name);
  alert("Hello " + name);
}

greet("Pixel Student");

Events

Use event listeners to respond to user actions.
document.getElementById("clickMe").addEventListener("click", function(){
  alert("Button Clicked!");
});

Change the page content dynamically using JS and innerHTML.
