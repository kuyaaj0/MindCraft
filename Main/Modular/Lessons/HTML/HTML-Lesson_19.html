HTML5 Media Elements

LESSON 19 - Cyberpunk Media Matrix

01. Video Element + Controls

video tag with controls = built-in play/pause/volume. Use source for multiple formats. 
<video width="600" controls>
    <source src="demo.mp4" type="video/mp4">
    <source src="demo.webm" type="video/webm">
    Your browser does not support video.
</video>

02. Audio Element

audio tag works exactly like video but no visuals. Multiple source tags for browser compatibility.

Click PLAY → Hear cyberpunk bell sound!

03. YouTube Embeds

Use iframe with YouTube URL. Add ?controls=0 to hide YouTube controls if needed. 
<iframe width="560" height="315" 
        src="https://youtube.com/embed/VIDEO_ID" 
        allowfullscreen></iframe>

04. Media Attributes

Common attributes: controls, autoplay, loop, muted, preload.
  • controls → Play/Pause/Volume bars
  • autoplay muted → Auto-play (required muted)
  • loop → Repeats forever
  • poster="image.jpg" → Video thumbnail

05. Fallback Text

Content inside tags = fallback text for old browsers. Shows when media fails to load. 
<video controls>
    <source src="video.mp4" type="video/mp4">
    <p>UPDATE YOUR BROWSER - HTML5 VIDEO NOT SUPPORTED</p>
</video>
