Table Structure

Lesson 1
A table is made of <table>, rows <tr>, and cells <td>. 
<table>
  <tr>
    <td>Cell 1</td>
    <td>Cell 2</td>
  </tr>
</table>
Cell 1 Cell 2

Table Headers

Lesson 2
Use <th> for header cells. 
<table>
  <tr>
    <th>>Name</th>
    <th>>Age</th>
  </tr>
</table>
Name Age

Simple Data Table

Lesson 3
Add multiple rows of data using <tr> and <td>. 
<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>25</td>
  </tr>
  <tr>
    <td>Bob</td>
    <td>30</td>
  </tr>
</table>
Name Age
Alice 25
Bob 30

Styling Table

Lesson 4
You can style tables using CSS, e.g., border colors, background, padding. 
<table style="border:2px solid green;">
  <tr style="background:#cfc;">
    <td>Styled Cell</td>
  </tr>
</table>
Styled Cell
