Input Attributes UX

LESSON 18 - Cyberpunk Validation Matrix

01. Placeholder Text

placeholder = gray hint text inside inputs. Disappears when user types.

02. Required Fields

required = browser blocks empty submits. Shows red validation automatically.
Cannot be empty!
10 digits only (0912345678)

03. Pattern Validation

pattern = regex for exact format matching. Use title="" for error message.
Format: 2024-IT-0001
4 digits only

04. Min, Max, Step

min/max = number limits.
