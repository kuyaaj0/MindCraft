LESSON 18

Lesson: Placeholder Attribute

Placeholder Attribute

What is a placeholder?

The placeholder attribute specifies a short hint that describes the expected value of an input field.

The placeholder is displayed in the input field until the user enters a value.

<input type="text" placeholder="Enter your name">

Required Attribute

What is the required attribute?

The required attribute specifies that an input field must be filled out before submitting the form.

If the user attempts to submit the form without filling out the required field, the browser will display a warning.

<input type="text" required>

Pattern Attribute

What is the pattern attribute?

The pattern attribute allows you to define a regular expression that the input value must match.

Use the pattern attribute to validate specific formats, such as email addresses or phone numbers.

<input type="text" pattern="^[A-Za-z]+$">

Min/Max Attribute

What are the min/max attributes?

The min and max attributes specify the range of acceptable values for an input field, typically used with number input types.

Ensure the entered number falls within the specified range.

<input type="number" min="1" max="10">

Step Attribute

What is the step attribute?

The step attribute specifies the legal number intervals for an input field, typically used with number inputs.

It helps define the precision or increment between allowed values.

<input type="number" step="0.5">
