1. DOCTYPE Declaration

The <!DOCTYPE html> must be the very first line. It tells browsers this is HTML5 and ensures proper rendering. Without it, browsers use "quirks mode" which breaks layouts.[web:1]

<!DOCTYPE html>

2. <html> Root Element

The <html> tag is the root container for all content. It wraps <head> and <body>. Use lang="en" attribute for accessibility.[web:1][web:4]

<html lang="en">
  
</html>

3. <head> Element

<head> holds metadata only: title, charset, viewport for mobile, links to CSS/JS. Nothing in <head> shows on the page.[web:1]

<head>
  <title>My Page</title>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>

4. <body> Element

<body> contains all visible content: text, images, links, etc. Exactly one per page.[web:1]
