Form Tags & Attributes

LESSON 15 - Cyberpunk Data Matrix

01. Form Tag Basics

The form tag creates input containers. Use action = where data goes, method = GET (URL) or POST (hidden). 
<form action="/submit.php" method="POST">
    <input name="username">
    <button>Submit</button>
</form>

02. Action & Method

action = server file URL. method="GET" = data in URL (search). method="POST" = data hidden (secure).
  • action="save-user.php" → sends to PHP file
  • method="GET" → ?name=john&age=20 (visible)
  • method="POST" → data hidden (passwords!)

03. Name Attribute

Every input needs a unique name. Server uses names to identify data (username, email, password).

Try typing → Submit → See data in URL (GET method demo)

04. Autocomplete & Required

autocomplete="on/off" → browser remembers inputs. required → blocks empty submits.

05. YOUR CHALLENGE!
